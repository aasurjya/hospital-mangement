# Context-First Guide

## Purpose
Use the markdown context layer before scanning the repository.

## Required Read Order
1. `md/ai-context.md`
2. `md/structure.md`
3. `md/automation.md`
4. `md/ai-clinical-assistant.md`
5. `md/pre-build-roadmap.md`
6. `.claude/knowledge/project-context.md`

## Working Rule
- Start with the markdown summaries.
- Use them to narrow the search scope.
- Only then inspect the relevant source folders.
- If the markdown context is stale, update it as part of the same task.
- For AI-related tasks, read `.claude/knowledge/ai-clinical-safety.jsonl` before inspecting implementation files.

## Expected Outcome
Lower search cost, faster onboarding for AI agents, and more consistent changes.
